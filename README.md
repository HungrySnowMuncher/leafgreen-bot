# leafgreen-bot
The goal of this project is to create an AI capable of beating Pokémon LeafGreen programmatically. The reason LeafGreen was chosen over other Pokémon games is:

1. Generation 1 is very buggy.
2. Generations 4 and 5 use 2.5D. Generations 6+ use 3D. It is more difficult to generate training data when the same game content can be viewed from many different angles.

Of the remaining options, I opted for Kanto's LeafGreen because the gameplay is the most straightforward.
